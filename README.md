# MENU 

## RUN DOCKER
```sh
$ docker build -t flask-form-app .
```

## RUN DOCKER
```sh
$ docker run -p 5000:5000 flask-form-app 
```


## COPY:
```sh
$ git clone https://github.com/Damir-p/FormWizard.git
```

## VIRTUALENV
```sh
$ python3 -m venv venv
```

## ACTIVATE VENV
```sh
$ source venv/bin/activate
```

## INSTALL REQUIREMENTS
```sh
(venv)$ pip install -r requirements.txt
```


## RUN
```sh
(venv)$ python app.py
```
